# 日语50音图演示

本示例呈现一个平假名 + 片假名的 50 音图，点击任一字符即可通过浏览器的语音合成播报对应发音。

## 快速开始
1. 到项目文件夹 `cd 下载目录/kana-chart/`
2. `python3 -m http.server 8000`
3. 在本地启动服务器并访问 `http://localhost:8000/kana-chart/`。
4. 确认浏览器允许语音合成（部分浏览器需在设置里启用“语音”或“屏幕阅读”权限）。
5. 拖拽或缩放并非此 demo 的功能；只需点击任一假名即可播放发音。

## 技术栈
- 纯静态资源：HTML + CSS + ES Module 模块。
- 使用 `SpeechSynthesisUtterance`（`ja-JP`）负责发音。
- 无需额外依赖程序包或构建步骤。
